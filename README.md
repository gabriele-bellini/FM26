# FM26

Artifact for the FM 2026 submission. This repository includes code, data, and instructions to reproduce the results.

## Content

This repository contains two main folders:

- `code`: Contains the protocol and fuzzer implementation in C++.
- `fuzzingExecutionTraces`: Contains two complete traces of fuzzer executions-one for the simplest scenario and one demonstrating extensibility.

## How to run ASMETA

You can download the ASMETA Eclipse environment to run `.asm` files here: [ASMETA GitHub](https://github.com/asmeta/asmeta).

References, documentation, and tutorials for ASMETA are available at: [ASMETA User Documentation](https://asmeta.github.io/userdoc.html)
